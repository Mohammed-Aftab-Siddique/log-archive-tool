# log-archive-tool
This is a simple script which can help you create a tarball.
You can edit `ARCHIVE_PATH` to make the tarballs save at your desired path.

# Installation
Clone the repository:
```bash
git clone https://github.com/Mohammed-Aftab-Siddique/log-archive-tool.git
```

Change directory:
```bash
cd log-archive-tool
```

Provide executible permission to the script file:
```bash
chmod +x log-archive-tool.sh
```

# Run the script
To run the script type `./linux-server-stats.sh` and enjoy.
